# Developer Usage

Smart Forms provides the following open-source TypeScript libraries:

1. **Smart Forms Renderer** ([@aehrc/smart-forms-renderer](https://www.npmjs.com/package/@aehrc/smart-forms-renderer))

   A React-based library that contains the rendering engine. It acts as a reference implementation for the [SDC Form Filler](https://hl7.org/fhir/uv/sdc/CapabilityStatement-sdc-form-filler.html).

2. **SDC Populate** ([@aehrc/sdc-populate](https://www.npmjs.com/package/@aehrc/sdc-populate))

   A reference implementation of the [SDC Populate](https://hl7.org/fhir/uv/sdc/OperationDefinition-Questionnaire-populate.html) operation, also known as $populate.
   Please refer to the [sdc-populate API](/docs/api/sdc-populate) for more information.

3. **SDC Assemble** ([@aehrc/sdc-assemble](https://www.npmjs.com/package/@aehrc/sdc-assemble))

   A reference implementation of the [SDC Assemble](https://hl7.org/fhir/uv/sdc/OperationDefinition-Questionnaire-assemble.html) operation, also known as $assemble.
   Please refer to the [sdc-assemble API](/docs/api/sdc-assemble) for more information.

4. **SDC Template Extract** ([@aehrc/sdc-template-extract](https://www.npmjs.com/package/@aehrc/sdc-template-extract))

   A reference implementation of the [SDC Template Extract](http://hl7.org/fhir/uv/sdc/OperationDefinition-QuestionnaireResponse-extract.html) operation, also known as $extract.
   Please refer to the [sdc-template-extract API](/docs/api/sdc-template-extract) for more information.
