# Date

[Question with a date answer (valueDate)](https://hl7.org/fhir/r4/codesystem-item-type.html#item-type-date).

It uses the FHIR [date](https://hl7.org/fhir/r4/datatypes.html#date) data type, which accepts YYYY, YYYY-MM, or YYYY-MM-DD as inputs, e.g. 2018, 1973-06, or 1905-08-23.
