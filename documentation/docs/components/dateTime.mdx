# Date Time

[Question with a date and time answer (valueDateTime)](https://hl7.org/fhir/r4/codesystem-item-type.html#item-type-dateTime).

It uses the FHIR [dateTime](https://hl7.org/fhir/r4/datatypes.html#dateTime) data type, which accepts YYYY, YYYY-MM, YYYY-MM-DD or YYYY-MM-DDThh:mm:ss+zz:zz as inputs, e.g. 2018, 1973-06, 1905-08-23, 2015-02-07T13:28:17-05:00 or 2017-01-01T00:00:00.000Z.
