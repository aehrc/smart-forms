# Time

[Question with a time (hour:minute:second) answer independent of date (valueTime)](https://hl7.org/fhir/r4/codesystem-item-type.html#item-type-time).

It uses the FHIR [time](https://hl7.org/fhir/r4/datatypes.html#time) data type.

**There are plans to convert this to be similar to the dateTime component's date field.**

:::warning

This component is not thoroughly tested yet. Please use it with caution.

:::
