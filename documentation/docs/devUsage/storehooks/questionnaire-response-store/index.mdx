# Questionnaire Response Store

The QuestionnaireResponseStore exposes the following properties:

- **sourceResponse**
- **updatableResponse**
- **updatableResponseItems**
- **formChangesHistory**
- **invalidItems**
- **responseIsValid**

Refer to the [API Reference](/smart-forms/docs/api/smart-forms-renderer/interfaces/QuestionnaireResponseStoreType) for more information on each property.
