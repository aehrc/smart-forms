# QuestionnaireStore

The QuestionnaireStore exposes the following properties:

- **sourceQuestionnaire**
- **itemTypes**
- **tabs**
- **currentTabIndex**
- **variables**
- **launchContexts**
- **enableWhenItems**
- **enableWhenLinkedQuestions**
- **enableWhenIsActivated**
- **enableWhenExpressions**
- **calculatedExpressions**
- **answerExpressions**
- **processedValueSetCodings**
- **processedValueSetUrls**
- **cachedValueSetCodings**
- **fhirPathContext**
- **populatedContext**
- **focusedLinkId**
- **readOnly**

Refer to the [API Reference](/smart-forms/docs/api/smart-forms-renderer/interfaces/QuestionnaireStoreType) for more information on each property.
