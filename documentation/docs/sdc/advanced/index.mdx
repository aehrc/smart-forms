# Advanced Rendering/Behavior

SDC provides a wide range of extensions to enhance how a Questionnaire items are rendered or control the behavior of the items:

- [Advanced Form Rendering](https://hl7.org/fhir/uv/sdc/rendering.html)
- [Form Behavior and Calculation](https://hl7.org/fhir/uv/sdc/behavior.html)

This section breaks down the supported extensions in Smart Forms into different subsections.

#### [Fundamental Elements](/smart-forms/docs/sdc/advanced/fundamental)

- required
- repeats
- readOnly
- hidden
- initial

#### [Group Rendering](/smart-forms/docs/sdc/advanced/group)

- gtable
- grid
- tab-container

#### [Display/Text Rendering](/smart-forms/docs/sdc/advanced/display)

- prompt
- unit
- lower
- upper
- flyover
- rendering-xhtml
- displayCategory
- openLabel
- shortText

#### [Question Rendering](/smart-forms/docs/sdc/advanced/question)

- autocomplete
- drop-down
- check-box
- radio-button
- slider
- choiceOrientation
- sliderStepValue
- entryFormat
- questionnaire-unit

#### [Value Constraints](/smart-forms/docs/sdc/advanced/constraint)

- required
- maxLength
- minLength
- regex
- minValue
- maxValue
- maxDecimalPlaces
